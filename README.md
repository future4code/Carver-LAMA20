- Node
- SQL
- MySQL

- Dotenv
- Knex
- React
- Axios
- UUID
- Bcrypt
- Hash
- json
- jest

## Os endpoints são:

   → Criar usuário
   
   → Logar usuário

   → Cadastrar banda

   → Pegar dados de uma banda pelo nome

   → Pegar dados de uma banda pelo id

   → Cadastrar show a um dia

   → Pegar todos os shows de um dia
